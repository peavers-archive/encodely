SOCAT

SOCAT is used to allow the docker-java library to easily talk to the docker service. Probably don't touch this command unless you really know what you're doing.

Containers

Prefix all containers with this value. Is used to start/stop containers so make sure other containers won't contain this. Note: If you change this while the SOCAT or other containers are running, you'll need to manually stop them before restarting the application.
